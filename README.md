#  Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using Python in Jupyter Notebook.

##  Project Overview

This project allows a user to play Rock Paper Scissors against the computer. The computer makes a random choice, and the program determines the winner based on the game rules.

##  Features

- User-friendly gameplay
- Random computer choices
- Winner determination logic
- Input validation
- Built using Python

## Technologies Used

- Python
- Jupyter Notebook

##  How to Run

1. Open the Jupyter Notebook file (`rock paper scissors.ipynb`).
2. Run all cells.
3. Enter your choice:
   - Rock
   - Paper
   - Scissors
4. View the result and play again.

##  Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choices result in a draw

##  Sample Output
Enter your choice (Rock/Paper/Scissors): Rock

Computer chose: Scissors

You Win!
